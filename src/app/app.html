Acerca de mi

Dentro de mi experiencia, desarrollé un sistema de inventario y facturación para el local de comida rápida Centaurus Fast Food, utilizando C# como tecnología principal. El sistema permitió gestionar productos, controlar el inventario en tiempo real y automatizar el proceso de facturación, optimizando la eficiencia operativa del negocio. Este proyecto me permitió aplicar buenas prácticas de programación, diseño de bases de datos y enfoque en la experiencia del usuario final.

Me considero una persona proactiva, orientada a resultados y con facilidad para aprender nuevas tecnologías, cualidades que me permiten integrarme a equipos de desarrollo y contribuir al crecimiento de las organizaciones.
